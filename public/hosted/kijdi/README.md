# 💬 ChatApp - تطبيق المراسلة الكامل

## 📁 هيكل الملفات

```
chat_app/
├── index.html      ← الواجهة الرئيسية (HTML + CSS + JS)
├── api.php         ← الـ API الخلفي (كل العمليات)
├── config.php      ← إعدادات قاعدة البيانات
├── database.sql    ← ملف قاعدة البيانات
├── uploads/        ← مجلد الصور والملفات (أنشئه يدوياً)
└── README.md       ← هذا الملف
```

---

## 🚀 خطوات التثبيت (XAMPP / WAMP / cPanel)

### 1. إعداد قاعدة البيانات
1. افتح phpMyAdmin
2. اضغط "استيراد" → اختر ملف `database.sql`
3. اضغط "تنفيذ"

### 2. إعداد config.php
افتح الملف وعدّل:
```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');      // ← اسم مستخدم MySQL
define('DB_PASS', '');          // ← كلمة مرور MySQL
define('DB_NAME', 'chat_app');  // ← اسم قاعدة البيانات
```

### 3. رفع الملفات
- ضع كل الملفات في مجلد داخل `htdocs` (XAMPP) أو `public_html` (cPanel)
- أنشئ مجلد `uploads/` وأعطه صلاحية كتابة:
  ```
  chmod 755 uploads/
  ```

### 4. فتح التطبيق
```
http://localhost/chat_app/
```

## 🛡️ الأمان
- كلمات المرور مشفرة بـ `password_hash()`
- الجلسات محمية بـ PHP Sessions
- جميع المدخلات منقاة من XSS
- استعلامات PDO المحضرة ضد SQL Injection
